PEPSI DISTRIBUTION MANAGEMENT SYSTEM
=====================================

REQUIREMENTS:
  - Node.js 22+  (https://nodejs.org)
  - Git          (https://git-scm.com)   — only needed for Sync / Updates

─────────────────────────────────────────────
 DEVELOPER PC  (where you edit code)
─────────────────────────────────────────────
1. Download or git clone:  git clone https://github.com/abbasinkhan567-beep/DMS-BY-AK-HK-AK.git
2. Double-click  START.bat     (first run installs packages + builds app)
3. Login:  admin123
4. Edit code in VS Code → test with START.bat
5. Publish updates:    PUBLISH.bat        (pushes code to GitHub)
6. Make Office folder: MAKE-OFFICE.bat    (creates C:\Pepsi-Office)

Files in this folder:
  START.bat           Launch the app
  PUBLISH.bat         Push code changes to GitHub
  MAKE-OFFICE.bat     Create ready-to-run folder for Office PC
  MAKE-SHORTCUT.vbs   Create desktop shortcut manually
  BACKUP-NOW.bat      Manual database backup
  version.json        Current version number

─────────────────────────────────────────────
 OFFICE PC  (runs the app — no coding)
─────────────────────────────────────────────
1. Copy the C:\Pepsi-Office folder from Developer (via USB / network / MAKE-OFFICE.bat)
   OR git clone on Office PC:  git clone https://github.com/abbasinkhan567-beep/DMS-BY-AK-HK-AK.git
2. Double-click  START.bat
3. Login:  admin123
4. Settings → Sync tab:
   - PC name:  Office
   - GitHub token:  paste your token  →  Save
   - Click  Sync Now     (data will sync between Developer + Office PCs)
5. New updates:  Settings → Updates → Apply

─────────────────────────────────────────────
 SYNC SETUP (for GitHub token)
─────────────────────────────────────────────
1. Go to https://github.com/settings/tokens  →  Generate new token (classic)
2. Check only:  repo (full control)
3. Copy the token → Settings → Sync tab → paste → Save → Sync Now
